（windows环境下，全程建议使用git bash命令行）

使用npm安装hexo

npm是nodejs的包管理器，一般在nodejs安装时已附带安装，安装nodejs后可直接在命令行中使用

初始化hexo博客目录

手动新建一个文件夹，在文件夹下git bash（或者cd + 目录名进入该目录），然后

生成以下文件

  • node_modules（项目依赖包）

  • public（存放生成页面）

  • scaffolds：命令生成文章等的模板

  • source：用命令创建的各种文章

  • themes：主题

  • _config.yml：整个博客的配置

  • package.json：项目所需模块项目的配置信息

    其中我们需要重点关注的是source、themes文件夹以及_config.yml文件

建立github仓库

登录github建立新仓库，注意命名形式为yourusername.github.io（yourusername是你的git账户）
